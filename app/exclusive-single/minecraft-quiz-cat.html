MC 是 3D 游戏，这里问的是 3D 的猫，而不是2D的贴图。要求形状和颜色都是轴对称的，但首先形状都没有轴对称，所以不用看颜色了，没有一种猫是轴对称的。注意区分三维空间的轴对称（又叫 180 度旋转对称）和镜像对称（又叫面对称、空间反演对称）。

已驯服的猫的实体数据中会记录哪一项？
出生地
出生时刻
驯服时刻
主人的 UUID
MC 中没有生物会记录自己的出生时刻和驯服时刻，只有海龟会记录自己的出生地，所有驯服的生物都会记录自己主人的 UUID 。只有当猫是玩家第一个驯服的生物时，驯服时刻才会记录到进度中，但不是实体数据中。现实世界的猫也不会记得日期之类的东西，所以铲屎官们一定要记好主子来到这个世界的日期以及签订契约的日期。但是猫会永远记得自己的主人。
哪一项和猫具有相同的命名空间 ID ？
豹猫
猫刷怪蛋
音乐唱片 cat
以上都不是
猫的命名空间 ID 是 cat ，豹猫的命名空间ID是 ocelot ，猫刷怪蛋的命名空间 ID 是 cat_spawn_egg ，音乐唱片 cat 的命名空间 ID 是 music_disc_cat 。其实不用查资料，既然它是 ID ，那么当然就是独一无二的。
村庄中生成猫的最低条件是？
1个渔夫
1个村民和4张床
2个箱子和2张床
1个钟
未驯服的猫会在至少有 1 个村民和 4 张床的村庄中生成。每 4 张有效床生成 1 只猫，最多可生成 10 只猫。

猫不可能受到哪一种伤害？
挤压伤害
来自溺尸的伤害
苦力怕的爆炸伤害
潮涌核心的魔法伤害
溺尸的三叉戟会扔偏，即使没有仇恨也能造成伤害；猫只是能够驱离苦力怕，它并不免疫苦力怕的伤害；潮涌核心仅对敌对生物造成伤害。
